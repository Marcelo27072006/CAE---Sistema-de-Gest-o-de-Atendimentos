# 🚀 CAE — Sistema de Gestão de Atendimentos

> Sistema web para gerenciamento de atendimentos do **Centro de Apoio ao Empreendedor (CAE)**, com foco na centralização de informações, rastreabilidade de atendimentos e apoio à tomada de decisão.

---

## 📌 Sobre o Projeto

O Centro de Apoio ao Empreendedor (CAE) realiza atendimentos voltados a pequenos empresários e pessoas que desejam iniciar um negócio, oferecendo serviços como consultorias, mentorias e capacitações.

O sistema foi desenvolvido para solucionar problemas identificados no processo atual, como:

- Descentralização de dados em múltiplas planilhas  
- Ausência de histórico estruturado de atendimentos  
- Dificuldade de comunicação entre equipes  
- Processo manual na geração de relatórios  

A solução proposta implementa um **MVP (Minimum Viable Product)** funcional, com todas as principais telas operacionais.

---

## 🧠 Arquitetura

O sistema adota uma arquitetura simples e eficiente, adequada ao escopo do projeto:

```text
Frontend → HTML + CSS + JavaScript  
Estilo → Interface moderna inspirada em sistemas corporativos (ex: Salesforce)  
Dados → Simulados em memória (front-end)
```

## 🖥️ Funcionalidades

### 📊 Dashboard
- Indicadores (KPIs) em tempo real  
- Listagem de atendimentos recentes  
- Sistema de notificações  
- Gráfico de atendimentos por período  
- Indicadores de desempenho  

---

### 👤 Empreendedores
- Cadastro completo com validação  
- Busca por nome ou CPF  
- Filtros por tipo (MEI, Microempresa, Autônomo)  
- Atualização dinâmica da listagem  

---

### 📋 Atendimentos
- Registro de novos atendimentos  
- Associação com empreendedor, consultor e serviço  
- Controle de status (pendente, em andamento, concluído)  
- Visualização detalhada em modal  

---

### 📚 Histórico
- Consulta por empreendedor  
- Timeline completa de atendimentos  
- Registro de anotações e encaminhamentos  
- Rastreabilidade das interações  

---

### 💬 Mensagens
- Comunicação entre atendentes e consultores  
- Lista de conversas  
- Indicadores de mensagens não lidas  
- Respostas simuladas  

---

### 📈 Relatórios
- Dados consolidados mensais  
- Serviços mais demandados  
- Desempenho por consultor  
- Indicadores visuais (gráficos)  

---

### 🤖 Assistente IA
- Integração com API externa (Claude, quando disponível)  
- Respostas inteligentes baseadas nos dados do sistema  
- Apoio à análise e tomada de decisão
